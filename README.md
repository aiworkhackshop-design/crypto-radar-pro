# Crypto Radar Pro

Crypto Radar Pro is a safe crypto market radar and paper-trading dashboard.

## Features

- CoinGecko market data
- Whale-radar style scoring
- Buy watch / danger / neutral classification
- Paper trading only
- No private keys
- No real trading
- No MEV
- No front-running
- No wallet connection
- Mobile-first dashboard

## Run locally

```bash
npm install
npm start
```

Open the app in your browser.

## Replit

Import this repository into Replit and press Run.

## Safety note

This project is not financial advice. It does not execute real trades. Always test with paper trading before considering any real-market integration.
